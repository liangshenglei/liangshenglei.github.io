1. 简介

MYSOL是一个关系型数据库管理系统，由瑞典MSOL AB 公司开发，属于 Oracle 旗下产品。MYSOL 是最流行的关系型数据库管理系统之一，在 WEB 应用方面，MysQL是最好的 RDBMS(Relational Database Management System,关系数据库管理系统)应用软件之-。

2. Sql语言

2.1 概念

SQL(Structured Query Language) 结构化查询语言，用于存取数据、更新、查询和管理关系数据库系统的程序设计语言。

  • 经验: 通常执行对数据库的“增删改查”，简称C (Create) R (Read) U (Update) D (Delete)。
